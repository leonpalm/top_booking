# 听海之音 - 现代极简民宿网站

## 项目概述

这是一个为现代极简风格民宿"听海之音"设计的展示和预定网站。网站采用移动优先的设计理念，具有优雅的视觉效果和流畅的用户体验。

### 网站地址
**访问地址**: https://w7aoxysbf67ai.ok.kimi.link

## 功能特性

### 🏠 主页 (index.html)
- **Hero区域**: 全屏背景图片展示民宿外观，配合动态粒子效果
- **民宿介绍**: 简洁的文字介绍和特色服务展示
- **空间预览**: 精选图片展示主要空间
- **客户评价**: 真实客户反馈展示
- **动态效果**: 文字分割动画、卡片悬浮效果、滚动触发动画

### 🖼️ 展示页 (gallery.html)
- **图片轮播**: 使用Splide.js实现响应式图片轮播
- **全屏查看**: 点击图片可全屏浏览
- **房间详情**: 详细展示每个房间的特色和设施
- **视频展示**: 预留视频播放功能
- **分类展示**: 按房间类型分类展示

### 📅 预定页 (booking.html)
- **多步骤表单**: 4步预定流程，清晰的用户引导
- **日期选择**: 支持入住和退房日期选择
- **房间选择**: 标准间和豪华套房两种选择
- **价格计算**: 实时计算总价格，支持额外人数费用
- **价格图表**: 使用ECharts展示价格趋势
- **表单验证**: 实时验证用户输入
- **预定确认**: 完整的预定信息确认流程

## 技术特性

### 🎨 设计特色
- **移动优先**: 专为移动设备优化的响应式设计
- **极简风格**: 大量留白，简洁的布局
- **温暖色调**: 米白色主色调配合蓝绿色强调色
- **优雅字体**: Playfair Display + Inter 字体组合

### ⚡ 交互功能
- **流畅动画**: 使用Anime.js实现页面过渡和元素动画
- **触摸友好**: 支持滑动、点击等移动端手势
- **实时反馈**: 表单验证和操作的即时反馈
- **无障碍设计**: 支持键盘导航和屏幕阅读器

### 📱 移动端优化
- **底部导航**: 移动端友好的底部固定导航栏
- **触摸优化**: 按钮和交互区域符合移动端标准
- **快速加载**: 图片懒加载和资源优化
- **适配性强**: 支持各种屏幕尺寸

### 🔧 技术栈
- **前端框架**: 纯HTML5 + CSS3 + JavaScript
- **样式框架**: Tailwind CSS
- **动画库**: Anime.js
- **图表库**: ECharts.js
- **轮播组件**: Splide.js
- **字体**: Google Fonts (Playfair Display + Inter)

## 部署说明

### 本地运行
1. 下载所有文件到本地目录
2. 在项目根目录运行本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 访问 `http://localhost:8000`

### 生产环境部署
网站已部署到云服务器，支持直接访问：
- **访问地址**: https://w7aoxysbf67ai.ok.kimi.link
- **部署方式**: 静态网站部署
- **服务器**: Linux云ECS兼容

### 自定义部署
1. 将所有文件上传到您的Web服务器
2. 确保index.html为入口文件
3. 配置服务器支持静态文件访问
4. 建议使用Nginx或Apache作为Web服务器

## 使用说明

### 管理员操作
- **内容更新**: 直接修改HTML文件内容
- **图片更换**: 替换相应图片文件
- **价格调整**: 在booking.html中修改价格参数

### 用户操作
1. **浏览网站**: 在移动设备或电脑上访问
2. **查看房间**: 在展示页浏览房间图片和详情
3. **预定房间**: 在预定页完成4步预定流程
4. **联系确认**: 提交后等待管理员联系确认
